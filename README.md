🔐 Password Strength Checker

📌 Description

This is a simple Password Strength Checker project developed using Python.

The program checks whether a password is Strong or Weak based on the following conditions:

- Minimum 8 characters
- At least 1 uppercase letter
- At least 1 number
- At least 1 special character

This project was created as part of the Python Programming Internship at Kodbud.


🚀 Features

✔ User password input
✔ Checks password length
✔ Detects uppercase letters
✔ Detects numbers
✔ Detects special characters
✔ Displays Strong or Weak password result


🛠 Technologies Used

- Python
- Regex ("re" module)


▶ How to Run

1. Install Python on your system
2. Save the project file
3. Open terminal or PowerShell
4. Run the Python file to check password strength


📷 Example Output

- Strong Password Example: "Hello@123"
- Weak Password Example: "hello"
